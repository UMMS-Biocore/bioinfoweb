Six weeks basic computing workshop

Week 1: Unix environment

Goals:

  1. Basic command line familiarity
  2. Introduction to some well-known unix commands and tools: more, less, grep etc.
  3. Changing standard input and output. Unix pipes.
  4. File compression tools: tar and gzip
  5. Network protocols: SSH
  6. vi esentials.
  7. awk and simple usage.
