Galaxy

Please use the following address for inhouse galaxy installation with your UMassMed credentials; galaxy.umassmed.edu. If you can not login*, please send an email to Help Desk to join the galaxy group. Please use the clip below for a general demonstration. If you have any questions or need modifications in the pipelines or any additional tools you want to use in the galaxy, please send an email to biocore@umassmed.edu.

*First time users need to apply for an account in the cluster. Then send an email to hpcc-admins to join galaxy group.

MGHPCC User account access:
- Users sign up for a new GHPC account - Click Here
- If you want to run UMassTools; Please run the script once /project/umw_biocore/bin/addKey.bash in the cluster. This script will add galaxy keys to your home. Galaxy then will be able to submit jobs using your username.
